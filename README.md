## Key Skills Practiced

- **Constrained prompting** — the RCCFA formula (Role, Context, Constraints, Format, Anti)
- **Schema design** — defining structure before generating content
- **Personal branding** — tagline, profile image, honest skills assessment
- **AI persona design** — boundaries, tone, decline rules
- **Content evaluation** — the 5-check review process
